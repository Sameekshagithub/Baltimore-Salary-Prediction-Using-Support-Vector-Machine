How Support Vector Regression Works

Understanding the primary model powering this prediction engine

RBF Kernel

The Radial Basis Function kernel maps salary features into higher-dimensional space to capture non-linear relationships between job title, agency, and experience.

Epsilon-Insensitive Loss

SVR ignores prediction errors smaller than ε (epsilon), producing a more robust model that resists outlier salaries in the Baltimore dataset.

Hyperparameters

Tuned with C=100 (regularization), γ=0.1 (kernel width), ε=0.1 (tube width). GridSearchCV recommended for production.

Feature Scaling Required

SVR is sensitive to feature scale. All inputs are normalized via StandardScaler before training — a critical preprocessing step.
